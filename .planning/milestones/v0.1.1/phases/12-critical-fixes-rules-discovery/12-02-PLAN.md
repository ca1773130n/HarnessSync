<task type="auto">
  <name>Task 1: Add rules directory discovery to SourceReader</name>
  <files>src/source_reader.py</files>
  <action>
    Add a new method `get_rules_files()` to SourceReader that returns `list[dict]`. Do NOT modify the existing `get_rules()` method (backward compatibility).

    1. Add a private helper `_parse_rules_frontmatter(content: str) -> tuple[dict, str]`:
       - Split content on YAML frontmatter delimiters (--- at start of file).
       - Parse the frontmatter for `paths:` and `globs:` keys (support both per research open question #3).
       - Handle three formats:
         a. Single string: `paths: src/api/**/*.ts` -> ["src/api/**/*.ts"]
         b. YAML list: `paths:\n  - src/components/**/*.tsx\n  - src/hooks/**/*.ts` -> ["src/components/**/*.tsx", "src/hooks/**/*.ts"]
         c. Inline list: `paths: [a, b]` -> ["a", "b"]
       - Return (frontmatter_dict, body_without_frontmatter).
       - If no frontmatter, return ({}, full_content).
       - Prefer `paths:` over `globs:` if both present.
       - Use regex, NOT PyYAML (no new dependencies needed).

    2. Add `get_rules_files()` method:
       ```python
       def get_rules_files(self) -> list[dict]:
           """Return list of rules from .claude/rules/ directories.

           Returns:
               List of dicts with keys:
               - path: Path to the .md file
               - content: Markdown body (after frontmatter)
               - scope_patterns: List of path patterns from frontmatter (empty if none)
               - scope: 'user' or 'project'
           """
       ```

       Implementation:
       a. If scope in ("user", "all"): Walk `self.cc_home / "rules"` with `rglob("*.md")`.
          For each .md file found, parse frontmatter, append dict with scope='user'.
       b. If scope in ("project", "all") and self.project_dir: Walk `self.project_dir / ".claude" / "rules"` with `rglob("*.md")`.
          For each .md file found, parse frontmatter, append dict with scope='project'.
       c. Sort results by path for deterministic output.
       d. Use `read_text(encoding='utf-8', errors='replace')` with try/except for OSError/UnicodeDecodeError.
       e. Use `self.cc_home` for user rules (NEVER hardcode `~/.claude` per MEMORY.md).

    3. Update `discover_all()` to include rules_files:
       Add `"rules_files": self.get_rules_files()` to the returned dict (alongside existing "rules" key).

    4. Ensure `from __future__ import annotations` is already present (it is, line 1).
  </action>
